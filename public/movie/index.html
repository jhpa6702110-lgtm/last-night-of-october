TMDB API 설정

실시간 영화 데이터를 제공받으려면 API 키 또는 토큰을 입력해 주세요. 입력하지 않거나 미인증 시 내장된 데모 데이터를 사용합니다.

API 키 발급 방법:

  1. TMDB(themoviedb.org) 회원가입 및 로그인
  2. 우측 상단 프로필 > '설정(Settings)' > 'API' 메뉴 이동
  3. 'API 키 요청' 클릭 후 개발자(Developer) 유형으로 신청서 작성
  4. 생성된 API 키 (v3 auth) 또는 읽기 액세스 토큰 (v4 auth)을 복사하여 아래에 입력
HTML5 CSS3 Variable Tailwind CSS Vanilla JS TMDB API

영화 정보는 TMDB API를 통해 제공됩니다.
